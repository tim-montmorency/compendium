# ATELIER — SCHÉMA DE NAVIGATION 

## Scénario

Vous intégrez le site d'une boulangerie. Le client vous donne une liste de pages, mais aucune structure. À vous de proposer une architecture logique.


## Pages

- Accueil
- Notre histoire
- Nos produits (pains, viennoiseries, pâtisseries)
- Commande en ligne
- Nos boutiques (3 emplacements)
- Recrutement
- Contact

## Consigne (17 min)

Outil : 
- Microsoft Whiteboard 
 [https://whiteboard.cloud.microsoft/](https://whiteboard.cloud.microsoft/) (1 board par équipe de 3-4) 

Tâches :

1. Organiser les pages en arborescence (10 min)

2. Identifier :

- Quelles pages dans la nav principale?
- Quelles pages sont secondaires (footer)?
- Y a-t-il des regroupements?

3. Dessiner le schéma sur Whiteboard (7 min)
