# ATELIER — SCHÉMA DE NAVIGATION 

## Scénario

Vous intégrez le site d'une boulangerie. Le client vous donne une liste de pages, mais aucune structure. À vous de proposer une architecture logique.


## Liste des pages

- Accueil
- Notre histoire
- Conditions d'utilisation
- Politique de confidentialité
- Nos produits
- Pains
- Viennoiseries
- Pâtisseries
- Pizzas
- Sandwiches
- Nos boutiques (3 emplacements)
- Montréal
- Laval
- Terrebonne
- Recrutement
- Nous joindre
- FAQ
- Commande en ligne


## Consigne (15 min)

**Format:**

En équipe de 2-3 personnes.

**Outil:**

- [Draw.io dans VS Code](https://marketplace.visualstudio.com/items?itemName=hediet.vscode-drawio) 

**Tâches:**

1. Installer Draw.io in VS Code
2. Créer un nouveau fichier schema.draw.io
3. Organiser les pages en arborescence (10 min)
4. Identifier :
    - Quelles pages dans la nav principale
    - Quelles pages sont secondaires (navigation du pied de page)?
    - Y a-t-il des regroupements?
    - Y a-t-il des pages qui pourraient être dans des menus déroulants?
    - Pensez à l'ordre logique des liens dans les menus de navigation
5. Dessiner le schéma dans draw.io dans VS Code (vous pouvez partir du fichier modèle fourni)

## Mise en commun (5 min)

- Retour avec le reste de la classe.
- Discussions
