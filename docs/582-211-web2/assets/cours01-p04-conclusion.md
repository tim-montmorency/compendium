# Récapitulatif du Cours 1

### Aujourd'hui, vous avez appris :

✅ Votre **nouveau rôle** d'intégrateur à l'ère de l'IA  
✅ Ce que l'IA fait **bien** et ce qu'elle fait **mal**  
✅ Comment **analyser du code généré** et l'améliorer  
✅ Comment **lire une maquette** et prendre des décisions CSS  
✅ La **politique IA Phase 1** et pourquoi elle existe
