# La galerie d'art

L'objectif de cet exercice est de mettre en pratique la notion de mise en page automatique (grille).

## Concept

Vous devez créer une mosaïque d'images avec un thème de votre choix (animes, animaux, films, nourriture, etc.).

La galerie doit être irrégulière. C'est-à-dire que certaines images prennent plus de place que d'autres sur la grille.

## Consignes

- [ ] Créer un _Frame_ de `1600 x 800`
- [ ] Configurer le _Frame_ avec une mise en page dynamique en grille de `4 x 4`
- [ ] Configurer les espacements entre les cases de `30px` ainsi qu'autour du _Frame_

- [ ] Choisir une thématique
- [ ] Télécharger plusieurs images relatives à cette thématique et les inclure dans votre projet Figma pour construire la mosaïque 

- [ ] Appliquer un joli fond au _Frame_
- [ ] Soyez créatifs
