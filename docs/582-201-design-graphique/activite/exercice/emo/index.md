# Emo

Le but de cet exercice est d'associer une police de caractères à une émotion.

Les images à utiliser sont les suivantes : 

* [Peur](./fear.png){download}
* [Colère](./angry.png){download}
* [Bonheur](./happy.png){download}
* [Tristesse](./sad.png){download}

## Consignes

- [ ] Dans figma, créer 4 frames, 1 par émotion.
- [ ] Trouver une police qui représente bien l'émotion et l'inscrire par dessus l'image.
