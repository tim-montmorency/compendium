# Importer des *Assets* du *Asset Store*
Unity a un magasin nommé le *Asset Store* qui nous permet d'ajouter des objets 2D, 3D, des scripts et des matériaux à nos projets. Bien que vous deviez créer tous les assets de vos projets, il pourrait arriver que vous ayez à télécharger des Assets depuis le magasin. Vous devrez premièrement, trouvez une ressource dans le magasin, puis ensuite utiliser le Package Manager de Unity pour importer les ressources dans votre projet. Pour ce faire, suivez la procédure suivante:      


## Unity 2022.3.62f3

- [ ] Dans Unity, dans la barre de menu en haut à gauche, appuyez sur Window, Asset Store. 
- [ ] Une page Web s'ouvrira, trouvez alors la ressource que vous souhaitez télécharger et appuyez sur celle-ci.
- [ ] Appuyez sur Add to my assets, acceptez les conditions, et revenez dans Unity.
- [ ] Cliquez à nouveau sur Window dans la barre de menu, puis choisissez Package Manager. Une fenêtre s'ouvrira. Celle-ci contient plusieurs éléments présents dans votre projet.
- [ ] En haut à gauche, il y a un menu déroulant nommé Packages:In Project. Appuyez sur celui-ci et sélectionnez My Assets dans la liste. Vous verrez alors les éléments que vous avez ajouté à vos Assets depuis le Asset Store. Sélectionnez le paquet que vous venez d'ajouter depuis le Asset Store. Un résumé s'affichera à droite. Appuyez sur Download.
- [ ] Une fois que c'est téléchargé, vous pourrez cliquer sur Import, cliquez ensuite sur Install/Upgrade (Si demandé), puis sur Import dans la nouvelle fenêtre qui s'ouvre.


## Unity 6000.3.4f1

- [ ] Dans Unity, dans la barre de menu en haut à gauche, appuyez sur Window, Package Management, Asset Store. 
- [ ] Une page Web s'ouvrira, trouvez alors la ressource que vous souhaitez télécharger et appuyez sur celle-ci.
- [ ] Appuyez sur Add to my assets, acceptez les conditions, et revenez dans Unity.
- [ ] Cliquez à nouveau sur Window dans la barre de menu, puis choisissez Package Management, Package Manager. Une fenêtre s'ouvrira. Celle-ci contient plusieurs éléments présents dans votre projet.
- [ ] Dans la barre de gauche, vous retrouvez différents onglets: In project vous montre les paquets qui sont installés dans votre projet. My assets vous permet de voir les paquets ajoutés depuis le Asset Store.
- [ ] Dans My Assets, sélectionnez le paquet que vous venez d'ajouter depuis le Asset Store. Un résumé s'affichera à droite. Appuyez sur Download après le nom du paquet. 
- [ ] Une fois que c'est téléchargé, le bouton Download changera et Import to project sera inscrit. Appuyez sur Import, cliquez ensuite sur Install/Upgrade (Si demandé), puis sur Import dans la nouvelle fenêtre qui s'ouvre.
