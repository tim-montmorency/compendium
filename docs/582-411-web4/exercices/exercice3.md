## Exercice pratique – Utilisation de l’éditeur de thème WordPress

### Objectif

Apprendre à modifier l’apparence et la structure d’un site WordPress à l’aide de l’éditeur de thème, sans écrire de code.

### Mise en place 

1. Connectez-vous à l’administration WordPress
1. Activez le thème Twenty Twenty-Five
1. Créer 2 pages avec le contenu qui vous intéresse
1. Allez dans Apparence → Éditeur

## Étape 1 – Modifier les styles globaux

* [ ] Cliquez sur Styles
* [ ] Ouvrez la section Couleurs
* [ ] Changez :
 * La couleur principale
 * La couleur d’arrière-plan
* [ ] Ouvrez la section Typographie
* [ ] Modifiez :
 * La police des titres
 * La taille du texte normal
* [ ] Cliquez sur Styles → Blocs → Bouton
* [ ] Changez la couleur et l’arrondi des boutons
* [ ] Enregistrez les modifications

## Étape 2 – Modifier l’en-tête du site 

* [ ] Allez dans Parties de modèles
* [ ] Ouvrez En-tête
* [ ] Ajoutez un bloc Logo du site
* [ ] Ajoutez un bloc Navigation
* [ ] Ajoutez un bloc Paragraphe
* [ ] Écrivez un slogan de votre choix
* [ ] Alignez les éléments horizontalement
* [ ] Changez :
 * La couleur de fond
 * L’espacement vertical
* [ ] Enregistrez

## Étape 3 – Modifier le pied de page

* [ ] Allez dans Parties de modèles
* [ ] Ouvrez Pied de page
* [ ] Supprimez les blocs inutiles
* [ ] Ajoutez un bloc Colonnes (3 colonnes)
* [ ] Dans la première colonne :
 * Ajoutez le nom du site
* [ ] Dans la deuxième colonne :
 * Ajoutez un texte libre
* [ ] Dans la troisième colonne :
 * Ajoutez un menu ou un texte
* [ ] Modifiez :
 * Couleur de fond
 * Taille du texte
* [ ] Enregistrez

## Étape 4 – Modifier un modèle de page

* [ ] Allez dans Modèles
* [ ] Ouvrez le modèle Page
* [ ] Ajoutez en haut du modèle :
 * Un bloc Couverture
 * Une image de votre choix
* [ ] Ajoutez un Titre
* [ ] Centrez le titre
* [ ] Ajustez les marges et l’espacement
* [ ] Enregistrez

## Étape 5 – Observer l’impact

* [ ] Créez une nouvelle page
* [ ] Ajoutez du contenu (texte + image)
* [ ] Publiez la page
* [ ] Affichez la page sur le site
* [ ] Observez :
 * L’en-tête
 * Le pied de page 
 * Les styles
 * La structure
