## Instructions

- [ ] Effectuer un fork du [Codepen de départ](https://codepen.io/tim-momo/pen/dPGQmvO)

### HTML

- [ ] Configurer le code du SVG pour de le préparer à être animé avec `createDrawable()`

### CSS

- [ ] Corriger le positionnement (`position`) de l'image avant de l'animer

### JavaScript

- [ ] Animer la voiture de sorte à ce qu'elle suive le path du svg
  - [ ] Lissage linéaire
  - [ ] Durée 5 secondes
  - [ ] En boucle
- [ ] Animer le path du svg à la même vitesse que la voiture
