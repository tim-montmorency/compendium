# Donkey Kong

Le but de cet exercice est de reproduire le résultat attendu avec des classes Bootstrap.

!!! warning "Important"

    Ne pas ajouter de nouveau HTML. Modifier uniquement les attributs `class=""` déjà présents dans le code.
