## Instructions

- [ ] Effectuer un fork du [Codepen de départ](https://codepen.io/tim-momo/pen/raxQpWj)
- [ ] Segmenter le texte du paragraphe en lettres individuelles
- [ ] Animer l'opacité des lettre avec un délai affecté par la fonction stagger
- [ ] Boucler l'animation 3 fois seulement
- [ ] Ajouter un `modifier` afin d'éviter une animation progressive de l'opacité
