## Consignes

- [ ] Trouve les coordonnées géographiques de ta maison !

- [ ] Effectuer un fork du [Codepen de départ](https://codepen.io/tim-momo/pen/myPRQZK)
- [ ] Créer une carte maplibre avec les coordonnées de ta maison
- [ ] Appliquer un zoom de 17
- [ ] Retirer les crédits
- [ ] Appliquer le style `toner-v2` (_vector style_) de [maptiler](https://www.maptiler.com/fr/)

En CSS 

- [ ] Ajoute à la carte ces styles : 

    ```css
    border-radius: 100%;
    mix-blend-mode: multiply;
    border: 9px solid black;
    ```
