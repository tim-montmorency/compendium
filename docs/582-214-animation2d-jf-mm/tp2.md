# TP2 – Animation d’une scène en parallaxe  
**(20% de la note finale)**

## CONSIGNE

Vous êtes approchés pour réaliser une animation courte en parallaxe qui présente une œuvre multimédia existante (jeu vidéo, film, série télévisée, etc.).  

Le contenu doit demeurer approprié à un cadre scolaire.

Le projet doit comprendre les demandes suivantes :

1. Préproduction  
2. Animation  
3. Livrable (remise)

---

## Préproduction

La préproduction doit comprendre :

### Storyboard

- Le nom et le type de l’œuvre  
- Une description du ton  
- Une description de l’intention (quel sentiment doit être ressenti ?)  
- Le découpage visuel de **3 plans**  
- Indication des mouvements de caméra ou des éléments  
- Indication sonore  

### Moodboard

- Références visuelles  
- Palette de couleurs  
- Typographie envisagée  

---

## Animation

L’animation doit comprendre les éléments suivants :

- Une durée entre **10 et 30 secondes**
- L’intégration d’une bande sonore (musique et/ou effets sonores)
- La présence de **3 plans distincts** et chaque plan doit comprendre :
  1. Un avant-plan
  2. Un plan intermédiaire
  3. Un arrière-plan
  4. Une animation en effet parallaxe (déplacement d’une caméra)
  5. Au moins une animation avec keyframe
  6. Au moins un masque (ou cache)
- Un titre animé (title reveal) 
- Un générique de fin avec les éléments suivants :
  - Le nom du réalisateur / de la réalisatrice
  - Le nom du cours avec le superviseur (enseignant)
  - La liste complète des sources pour les images et le son
  - Les noms des TTP et autres en remerciements
  - Le logo TIM

---

## Livrable

Le tout doit être remis dans un dossier à votre nom avec les éléments suivants :

- Animation finale en format `.mp4`
- Tous les fichiers de travail (PSD, AEX, PRPROJ, etc.)
- Le storyboard
- Le moodboard
