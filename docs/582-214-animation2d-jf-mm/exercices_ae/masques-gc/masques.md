# Masques 

[Dossier de départ :material-download:](./masque.zip){ .md-button .md-button--primary }

## Ajout de masque

- [ ] Téléchargez le Dossier de départ
- [ ] Ouvrir After Effets et importer l'image Cadre
- [ ] Ajouter un calque solide noir par dessus
- [ ] Cliquer sur le calque solide, puis double cliquer sur l'outil ellipse pour faire un masque
- [ ] Dans les propriétés du masque, choisir soustraction
- [ ] Ajuster le Contour progressif et l'Expension du masque
- [ ] Ajuster l'opacité du calque pour faire une vignette
- [ ] Ajouter l'image Arbre derrière le cadre 
- [ ] Choisir l'outil plume et tracer le contour du cadre 
- [ ] Dans les propriétés du masque, choisir soustraction pour révéler l'arbre
- [ ] Ajuster l'Expension du masque
- [ ] Ajuster l'échelle et la position du calque de l'arbre
- [ ] Ajouter des ajustements comme courbes et/ou vibrance au besoin
- [ ] Fusionner tous les calques en faisant une pré-composition nommée : Background
- [ ] Garder le projet ouvert

## Ajout d'un cache 

- [ ] À partir du même projet, ajouter un calque de texte et écrire un mot (ex: Magie)
- [ ] Choisir la typo Bebas Neue
- [ ] Appliquer la couleur noir, ajuster la taille très grande (ex: 600px) et aligner au centre horizontal et vertical    
- [ ] Appliquer le cache sur la pré-composition Background
- [ ] Supprimer le calque de texte 
- [ ] Ajouter l'image Négatif devant la pré-composition Background
- [ ] Appliquer le cache en mode Luma sur la pré-composition Background
